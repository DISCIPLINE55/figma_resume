# Mentorship Resume

## Freelancer Portfolio

This repository contains the source code for a responsive freelancer portfolio website. The site showcases a freelancer's skills, services, creative work, and pricing, complete with a call-to-action to get started.

## Features

Responsive Design: The website is designed to look great on various devices, from desktops to mobile phones, thanks to a well-structured CSS layout.

## Hero Section

A prominent hero section introduces the freelancer, Shamima Akhi, and includes a brief description and a "Get Started" button.

## Stats Section

 Highlights key achievements and statistics, such as positive feedback, projects completed, and years of experience.

## Benefits & Services Section

 Dedicated sections explain the benefits of working with the freelancer and detail the various services offered, including web design, illustration, and SEO.

## Portfolio Showcase

 A gallery grid displays some of the freelancer's creative work.

## Pricing Plans

 A clear and simple pricing table presents different service packages (Basic, Premium, Super) with their respective features.

## Subscription Form

 A section to encourage visitors to subscribe for updates.

## Footer

 A comprehensive footer with brand information, social media links, and additional navigation links.

## Back to Top Button

 A convenient button for quick navigation back to the top of the page.

## Technologies Used

HTML5: For the website's structure and content.

CSS3: For styling and layout, including a modular approach with separate stylesheets for different sections.

Font Awesome: A popular icon library used for social media icons and service card icons.

Google Fonts (Inter, Ubuntu): A modern sans-serif font used for typography.

JavaScript: For dynamic functionality, such as the hamburger menu and the "Back to Top" button (as indicated by the script tag).

## How to Run Locally

To run the project locally, follow these steps:

## Clone the repository

Bash

git clone https://github.com/your-username/freelancer-portfolio.git

## Navigate to the project directory

Bash

cd freelancer-portfolio

### Open index.html

Simply open the index.html file in your preferred web browser.

### Customization

Content: All text content, including the freelancer's name, bio, and service descriptions, can be changed directly in the index.html file.

Images: Replace the placeholder images in the images/ directory with your own and update the src attributes in index.html.

Styling: Modify the CSS files in the css/ directory to change the color scheme, fonts, and layout to match your personal brand.

## Functionality

Update js/scripts.js to add or modify any interactive features
